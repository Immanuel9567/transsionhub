Tips and Guides

To Hide Phh Settings via Terminal preferably Termux, ROOT IS REQUIRED | su -c "pm disable me.phh.treble.app/.TopLevelSettingsActivity"

To Show Phh Settings, ROOT IS REQUIRED | su -c "pm enable me.phh.treble.app/.TopLevelSettingsActivity"

In case the prop doesn't already exist, use Terminal to set it. A possibke fix for laggy UI on A13+ systems (This might result in consistently high battery temps and consumption).

Run "su" in terminal and run setprop debug.sf.disable_backpressure 1" and Reboot
